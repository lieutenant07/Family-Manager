Welcome in Family Manager!

This is a demo application which keeps track of the family data. There are three main features which allow you to manage family data. First one serves the purpose of creating new family and add members such as fathers and subsequent children. Second feature allows you to preview previously added entities in seperate view. The last piece of functionality provides a search engine to look up families based on child characteristics.

This application has been created in response to preliminary task assigned in order to evaluate basic JAVA/Angular programming skills.
